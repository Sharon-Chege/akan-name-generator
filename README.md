<h1><strong>AKAN NAME GENERATOR</strong></h1>
<hr>

<h3>About the Project</h3>
<hr>

<p><h4>The <strong> Akan Name Generator</strong> will calculates the day of the week you were born and assigns them an Akan name based on their gender. A fun and interactive Name generating website built with HTML, CSS and JavaScript.This project demonstrates DOM manipulation, event handling, and basic game logic in a clean, responsive UI.</h4></p>
<hr>

<h3>Getting Started</h3>
<h4>Requirements</h4>
<h4>
    <ul>
     <li>Google Chrome</li>
     <li>Mozilla Firefox</li>
     <li>Microsoft Edge</li>
    </ul>
</h4>
<hr>

<h3>Setup Instructions</h3>
<h4>
<li>
    1. Clone repository
    https://github.com/Sharon-Chege/akan-name-generator

    2.Navigate to the project directory
    cd-akan-name-generator

    3.Open the game Simply open the index.html file in your preferred browser:
<ul>
    <li>Double-click index.html, OR </li>
    <li>Right-click → Open with... → select your browser</li>
</ul>
</li>
</h4>
<hr>

<h3>How to Play</h3>
<p><h4>The user enters their date of birth, the month they were born, the year they were born and their gender. The computer in turn calculates the day of the week they were born and returns the Akan Name together with the Day of the Week they were born.</h4></p>
